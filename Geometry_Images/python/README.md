## Geometry Images

### Algo: Cut path finder

#### Env

> `Python 3.9.6`

```bash
pip install -r requirements.txt
```

#### Run

```bash
cd Geometry_Images/python/

python3 run.py ../static/dragon1K.m

```

#### Interact

- press `enter` to cut triangles
- press `space` to cut edges (after triangles cleared)
- press `q` to quit
